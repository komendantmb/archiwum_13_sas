# Import źródeł 2026-04-21

Katalog zawiera import źródeł aktualnie dostępnych w projekcie.

Zasada techniczna tego importu:
- pliki tekstowe `.txt` zostały zapisane jako nośniki tekstowe
- pliki PDF zostały zarejestrowane przez manifest importowy
- dla binariów zachowano identyfikowalność przez nazwę oryginalną, nazwę techniczną, rozmiar i sumę SHA-256
- surowe binaria PDF nie zostały skopiowane do repo w tej operacji z uwagi na wolumen zbioru i ograniczenia zwykłego repo GitHub
