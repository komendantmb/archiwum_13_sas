# IPN Jan — wnioski wysłane

Folder na kopie pism wysłanych do IPN dotyczących Jana Stanisława Baczyńskiego.

Zasada nazewnictwa:
- RRRR-MM-DD_typ_pisma_adresat.md
- do każdego pisma można dodać skan potwierdzenia nadania albo UPP

Minimalne metadane przy każdym piśmie:
- data wysyłki
- adresat
- kanał wysyłki
- numer nadania
- status odpowiedzi
