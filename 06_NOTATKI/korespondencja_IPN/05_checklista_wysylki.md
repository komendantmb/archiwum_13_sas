# Checklista wysyłki pakietu IPN

## Pakiet podstawowy

- [ ] oficjalny formularz IPN dla zmarłej osoby najbliższej
- [ ] odpis aktu zgonu
- [ ] dokument potwierdzający pokrewieństwo albo status osoby najbliższej
- [ ] kopia dokumentu tożsamości, jeżeli wymagana w danym trybie
- [ ] `01_zalacznik_podstawa_wpisu.md`
- [ ] `02_zalacznik_teczki.md`
- [ ] `03_zalacznik_brakowanie.md`

## Pakiet dodatkowy

- [ ] `04_wyjasnienie_art_35b.md`
- [ ] własne załączniki rodzinne lub dokumenty uzupełniające
- [ ] własna notatka z numerem przesyłki i datą nadania

## Kontrola formalna przed wysyłką

- [ ] wszystkie daty i nazwiska sprawdzone
- [ ] wszędzie wpisano pełne brzmienie: Jan Stanisław Baczyński
- [ ] brak twierdzeń wykraczających ponad obecny materiał [P]
- [ ] pisma są podpisane
- [ ] tryb złożenia zgodny z aktualną kartą usługi IPN

## Kontrola po wysyłce

- [ ] zapisano numer nadania
- [ ] zapisano datę doręczenia
- [ ] utworzono numer odpowiedzi IPN po wpływie
- [ ] odpowiedź rozpisano na: co potwierdzone, czego brak, o co dopytać dalej
